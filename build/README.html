How to use

First, extract the distro zip file, then double-click index.html, or click this link to go there.

Next, create a key pair. Select "I want to: Generate a public-private key pair". Then, press "Generate Public-Private Key Pair". Two buttons named "Download public key" and "Download private key" should show up under "Output". Download both of them, and keep them.

If you want somebody to send you a message, send them the public key. Don't worry, as messages encrypted with the public key can only be decrypted by the private key, which means that you must keep the private key safe.

Remember, Share the PUBLIC key, Keep SECRET the PRIVATE key.

To encrypt a message, select "I want to: Encrypt an outbound message". Then, click "Choose File" under "public key of the reciever", then press "Load Public Key". It's successful when the filename shows up in green text.

How to modify

An archive containing the TypeScript/JavaScript source code is found within the distribution folder. Everything else is it's original source code.
